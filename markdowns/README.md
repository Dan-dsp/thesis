# Bird Species Classification Thesis

This folder contains the documentation for the top-level scripts in `01_Python`.

## Documentation map

- `INDEX.md` lists every documented Python file and its pipeline role.
- `sl_pipeline_overview.md` explains the current shallow-learning workflow in depth.
- `*_guide.md` files are the existing detailed shallow-learning guides.
- A Markdown file named after each `.py` file gives its purpose, inputs, outputs, and current status.

Only files directly inside `01_Python` are documented here. Subfolders are intentionally outside this documentation pass.
